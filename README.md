# AlarmQR
Mobile app that makes waking up easier.

Goal of AlarmQR is to force the user to get out of bed, instead of taking another nap.

Set up your alarm and bind it up with a chosen QR code. When you turn off the alarm, you have to scan the code to stop the countdown to next, unpleasent alarm, which can be turned off only by scanning the code. Place the QR code in such a place at home, that will force you to leave the bedroom and start your day.
